## Quick Reference

### Main UI Elements

| Element | Purpose |
|---------|---------|
| Floor Navigator | Switch viewing level |
| Block Picker | Select blocks to place |
| Info Panel | Block/resident details |
| Budget Panel | Financial overview |
| Time Controls | Pause, speed |
| Overlays | Toggle info layers |
| News Feed | Events and stories |

### Key Panels

- **Block Info** - Click any block for details
- **Resident Info** - Click resident for profile
- **Budget** - Monthly income/expense
- **AEI Dashboard** - Win condition progress
